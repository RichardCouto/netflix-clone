Clone Netflix (feito com a mentoria do Pasquadev)

Esse é o primeiro projeto do meu portfólio, as tecnologias usadas para desenvolver o projeto foram:

Linguagens:
- Java Script
 
Frameworks:
- React

API:
- The Movie DB

O projeto consiste em um clone da netflix, onde exibimos os cards dos filmes, organizados por categorias. Ao clicar em um filme você pode visualizar o trailer dele, caso esteja disponível e ao clicar novamente no filme o trailer se fecha.

PARA ACESSAR O PROJETO ONLINE, ACESSE O LINK: https://netflix-clone-6a8a4.web.app/
